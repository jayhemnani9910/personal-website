Keyboard-first speedcubing timer with Three.js 3D visualization, WCA-compliant scrambles for all puzzle sizes (2x2–7x7), rich statistics, session management, and full PWA offline support. Built as an 18-module vanilla JavaScript application with the Ember Glass design system.
