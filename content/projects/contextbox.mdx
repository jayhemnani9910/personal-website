# Overview
CLI-first personal knowledge assistant built with Python. Captures screenshots, extracts text via Tesseract OCR, embeds content with sentence-transformers into SQLite, and enables semantic search + AI-powered Q&A via GitHub Models. 27 Python modules with 6 content extractors, cross-platform support, and a React documentation frontend deployed on GitHub Pages.
